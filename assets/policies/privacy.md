# Kebijakan Privasi Andrew

Tanggal berlaku: 2026-01-09

1. Data yang Dikumpulkan
   - Profil: nama, fokus utama, target waktu.
   - Rencana/tugas, progres Pomodoro.
   - Data teknis: log kesalahan, perangkat (anonim).

2. Penggunaan Data
   - Menyimpan dan menyinkronkan rencana/profil.
   - Menyediakan notifikasi dan statistik fokus.

3. Penyimpanan & Keamanan
   - Disimpan lokal (Hive/SharedPreferences) dan/atau server (mis. Supabase) saat login.
   - Diupayakan perlindungan sesuai praktik wajar.

4. Berbagi Data
   - Tidak dijual; hanya dibagikan ke penyedia infrastruktur (mis. Supabase) sesuai kebutuhan layanan.

5. Hak Pengguna
   - Akses, ubah, hapus data rencana/profil.
   - Nonaktifkan sinkronisasi dengan keluar akun.

6. Retensi
   - Data lokal tersimpan di perangkat; data server dihapus saat akun dihapus sesuai kebijakan penyedia.

7. Perubahan
   - Akan diberi tahu jika ada perubahan penting.

8. Kontak
   - privacy@example.com
